    This class defines an element which is going to be stored in BTree.

    This class defines an element which is going to be stored in BTree. Each Node of BTree consists of NodeElements. NodeElement is a key-value pair. Each value is kept "under" its key. Keys are unique i.e. if inserting at a key which already exists then the value is being replaced.
